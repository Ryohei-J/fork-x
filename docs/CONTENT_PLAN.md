# コンテンツ拡充計画（Google AdSense審査対応）

Google AdSenseの審査通過とその後の集客のため、コンテンツ（ページ）を拡充する計画。
現状は計算機能（トップページ）と免責事項ページのみで、審査に必要な構造ページ・十分な文章量のオリジナルコンテンツが不足している。
